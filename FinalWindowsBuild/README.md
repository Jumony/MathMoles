## Build Instructions

### **Prerequisites**
- **Unity 6000.2.10f1**

### **Steps**
1. Extract the files from the ZIP into your desired directory.  
2. Open **Unity Hub**.  
3. In the *Projects* tab, click **Add project from disk**.  
4. Select the folder you extracted earlier.  
5. Open the project in Unity.  
6. Navigate to **File > Build and Run**.  
7. Choose your desired build output location.

---

## Run Instructions
- Double-click the provided executable `MathMoles.exe`.
